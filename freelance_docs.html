Add new Job

listing.add(title, price, rating, user, body);

title - a string, the title
price - a number 1 - 5 dictating price
rating - a number 1 - 5 dictating rating
user - a string, the name of the poster
body - the message of the post

id - NOT YET CREATED: will be used to open link, maybe more

Add new Option

options.add(type, name, values);

type - can be either "check" for checkboxes or "radio" for radio buttons
name - The name of option. "Sex" or "Masturbatory Tactics", etc. The ":" is added automatically.
values - An array of the values, best dictated like so: ["Man","Woman","Blob","Bot","Bitch"]
